# boilerplate
Personal boilerplate that makes use of my favourite stack

#### Stack
- Vue (vuex, vue-router)
- Vuetify
- TypeScript
- Pug
- Sass
- i18n

##### Project setup
```

##### Compiles and hot-reloads for development
```
npm run serve

```

```
